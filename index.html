Simon Say's Game

Simon Says Game Rules:

    Start the Game:
  • Press any key or click the "Start Game" button to initiate the game.
  • The game starts at Level 1.
    Game Sequence :
  • Simon will flash a random colored button in a sequence.
  • Watch and remember the sequence as it will be your reference.
    User's Turn :
  • After the game sequence is displayed, it's the user's turn to repeat the sequence.
    Repeat the Sequence :
  • Click on the colored buttons in the same order as displayed by Simon.
  • The sequence starts with one color and progressively gets longer with each successful round.
    Level Up :
  • If the user successfully repeats the sequence, the level increases.
  • Simon will add a new color to the sequence.
  • The user must then replicate the longer sequence.
    Game Over :
  • If the user makes a mistake and clicks the wrong button or fails to replicate the sequence correctly, the game is over.
  • The background turns red momentarily, indicating an incorrect move.
    Score :
  • The score is calculated based on the level reached.
  • Each level passed contributes to the score (e.g., level 3 = score of 30).
    Restart the Game:
  • After the game is over, the player can restart by clicking the "Start Game" button again.
Note: The game incorporates visual and auditory cues, with buttons flashing to indicate the sequence. Pay close attention, remember the sequence, and try to reach the highest level possible to achieve a higher score.
